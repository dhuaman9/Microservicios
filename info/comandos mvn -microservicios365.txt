****comandos de java para ejecutar los jars.

Se debe ejecutar en el siguiente orden:
java -jar target/ms365-configserver-0.0.1-SNAPSHOT.jar

java -jar target/ms365-eurekaserver-0.0.1-SNAPSHOT.jar

java -jar target/ms365-adminserver-0.0.1-SNAPSHOT.jar (opcional)

java -jar target/ms365-zuulserver-0.0.1-SNAPSHOT.jar

Posteriormente ejecutar  los microservicios de consumo, en cualquier oden, se sugiere por ejemplo:

java -jar target/ms365-usuarios-0.0.1-SNAPSHOT.jar

java -jar target/ms365-proyectos-0.0.1-SNAPSHOT.jar

java -jar target/ms365-proyectos-archivos-0.0.1-SNAPSHOT.jar
